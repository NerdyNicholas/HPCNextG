# fapolicyd Recent Denies

## Context
Quickly view recent fapolicyd blocks.

## Prompt
You are a Linux sysadmin.

Goal: Generate short commands to view recent fapolicyd denies.

Constraints:
- Output only commands
- Keep them short
- Assume journalctl available

Provide:
- 2–3 one-liners to:
  - view recent denies
  - filter relevant entries

If you output anything other than commands, you have failed.
