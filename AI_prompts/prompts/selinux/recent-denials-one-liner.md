# SELinux Recent Denials One-Liner

## Context
Need quick visibility into recent SELinux denials.

## Prompt
You are a senior Linux sysadmin.

Goal: Generate minimal one-liner commands to quickly identify recent SELinux denials.

Constraints:
- Output ONLY commands
- No explanations unless critical (1 line max)
- Keep commands short and practical
- Assume RHEL-like system

Provide:
- 2–3 one-liners to:
  1. Show recent denials
  2. Summarize causes
  3. Suggest quick triage view

If you output anything other than commands, you have failed.
